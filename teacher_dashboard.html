Common Questions

Can I restrict access?

Yes. You can control which teacher sees which batch's data. A Chemistry teacher won't see Physics records.

Does it work on Mobile?

Absolutely. We have a dedicated Teacher App (Android/iOS) for managing everything on the go.

Is there a storage limit?

No. Teachers can upload unlimited study material, assignments, and recorded lectures.

Can I track teacher attendance?

Yes, the Admin Dashboard allows you to track staff login times and class durations.
